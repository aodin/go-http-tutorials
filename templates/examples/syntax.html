Template Syntax Examples

Some examples of using the correct syntax in Go templates.

{{/* Comments will not show up in the HTML! */}}

Iterating a Map

A call to range with a single variable ( range $item := .Dict ) will return the value of each map element.

{{ if .Dict }} {{ range $item := .Dict }}{{ end }}
{{ $item.Name }} {{ $item.Value }} {{ $item.Status }}
{{ end }}

Iterating a Map with Key

A call to range with two variables ( range $key, $item := .Dict ) will return the key and value of each map element.

{{ if .Dict }} {{ range $key, $item := .Dict }}{{ end }}
{{ $key }} {{ $item.Name }} {{ $item.Value }}
{{ end }}

Iterating an Array

A call to range with a single variable ( range $item := .List ) will return each element of an array.

{{ if .List }}
    {{ range $item := .List }}
  1. {{ $item.Name }} ({{ $item.Value }})
    2. {{ end }}
{{ end }}

Iterating an Array with Index

A call to range with two variables ( range $index, $item := .List ) will return each index and element of an array.

{{ if .List }} {{ range $index, $item := .List }}{{ end }}
{{ $index }} {{ $item.Name }} {{ $item.Value }}
{{ end }}

A Note about Private Variables

A template variable that accesses a private attribute is valid syntax, but the template output will stop when it is encountered.

{{ if .Dict }} {{ range $item := .Dict }}{{ end }}
{{ $item.Name }} {{ $item.Value }} {{ $item.Status }} {{ $item.private }}
{{ end }}
